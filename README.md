## About

The Day Rave is a daytime wellness event in Amsterdam combining cold plunge, sauna, HIIT, Pilates, Booty & Core, Zumba, and a five-hour DJ-led coffee rave. No alcohol. No hangovers. Just movement, contrast therapy, and good music.

- **Date:** Sunday, 14 June 2026
- **Time:** 12:00 — 17:00
- **Venue:** IJland, Amsterdam Noord
- **Capacity:** 400 spots

## Deploy

### Netlify (drag & drop)
1. Go to [netlify.com](https://netlify.com)
2. **Add new site → Deploy manually**
3. Drag the project folder into the upload zone
4. Live in seconds

### GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Source: **Deploy from branch → main → / (root)**
4. Live at `https://yourusername.github.io/the-day-rave`

### Netlify + GitHub (auto-deploy)
1. Push to GitHub
2. On Netlify: **Add new site → Import from GitHub**
3. Select this repo — every push auto-deploys
